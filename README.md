## Install

**⚠️ Requires node.** [How to install node](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

**Method 1: using npm (recommended)**

`npm i -g git-ls`

or

**Method 2: using npx (supported)**

`npx git-ls`
