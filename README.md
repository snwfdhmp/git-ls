# git-ls: list directories and show git status

![git-ls demo](https://i.imgur.com/672nita.png)

## Install

**⚠️ Requires node.** [Tutorial here](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

**Using npm (recommended)**

`npm i -g git-ls`

or

**Using npx (supported)**

`npx git-ls`
